# ESP32-C3 LCD 1.28" Clock & Weather Display

A simple and lightweight project using **ESPHome** and **Home Assistant** to turn an **ESP32-C3-LCD-1.28** board into a smart display showing:

- 🕒 Real-time clock
- 🌤️ Weather conditions
- 🔄 Dynamic UI using LVGL
- 📟 Touch interaction (optional)

---

## 📦 Hardware

- ESP32-C3-LCD-1.28  
- 1.28" IPS display  
- 240x240 resolution  
- Capacitive touch support  
- USB-C power
